# Cripto Converter
Aplicativo converte valores de criptomoedas para real "BRL". Salva as conversões quando solicitado, a é
conversão realizada após o consumo da [API Mercado Bitcoin](https://www.mercadobitcoin.net/), 
que entrega o valor que criptomoeda está valendo no momento.
